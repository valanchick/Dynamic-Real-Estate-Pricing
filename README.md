## Бизнес-цель
Цель этого проекта — разработка **алгоритма умного ценообразования (Smart Pricing)** для рынка недвижимости и краткосрочной аренды (на примере датасета Airbnb NYC). 

Модель предсказывает справедливую рыночную стоимость (Fair Market Value) объекта на основе его геоданных, текстового описания и категориальных признаков. Это помогает арендодателям избегать занижения цен (потеря маржи) или их завышения (потеря заполняемости), что напрямую максимизирует оборот (GMV) платформы.

## Ключевые инженерные и аналитические решения

В этом проекте я вышла за рамки стандартных вызовов `.fit()` и `.predict()`, применив продвинутые (Senior-level) практики Data Science:

* **Работа с геоданными (Spatial Engineering):** Переход от моноцентрической к **полицентрической модели**. Извлечение точных координат с помощью векторизованной формулы Гаверсинуса для расчета `distance_to_center` (Таймс-Сквер) и `distance_to_ocean`, что позволило уловить сложную пространственную динамику цен.
* **Пространственная кластеризация (Zero Data Leakage):** Применение алгоритма `K-Means` для группировки объектов в скрытые сегменты качества. **Критически важно:** кластеризация и `StandardScaler` обучались строго на тренировочной выборке (X_train), чтобы избежать утечки данных (Data Leakage) при валидации.
* **Непараметрическое статистическое тестирование:** Математическая проверка продуктовых гипотез до начала машинного обучения. Использование **критерия Краскела-Уоллиса** для распределений (типы жилья) и **корреляции Спирмена** для доказательства "эффекта масс-маркета" (обратная связь цены и количества отзывов).
* **Робастное моделирование (MAE Loss):** Данные по недвижимости содержат экстремальные выбросы (например, пентхаусы за $10,000/ночь). Стандартная метрика `RMSE` сильно штрафовала бы модель и искажала веса. Обучение `CatBoostRegressor` было оптимизировано под функцию потерь **MAE (Mean Absolute Error)** (предсказание медианы), что сделало алгоритм устойчивым к аномалиям.

## Архитектура пайплайна

Пайплайн полностью реализован в едином Jupyter Notebook для удобства воспроизведения:
1. **EDA и проверка гипотез:** Визуализация геоданных, логическая импутация 20%+ пропусков, поиск пространственных аномалий.
2. **Feature Engineering:**
   * **NLP:** Извлечение семантических флагов (`is_luxury`, `has_view`) из названий объявлений с помощью регулярных выражений.
   * **Временные признаки:** Расчет "свежести" объявления (`days_since_last_review`).
3. **Target Encoding:** Использование встроенного в CatBoost регуляризованного target-кодирования для работы с признаками высокой кардинальности (более 200 уникальных микрорайонов).
4. **Оптимизация гиперпараметров:** Строгая кросс-валидация (3-Fold CV) с использованием `GridSearchCV`.

## Результаты и бизнес-инсайты

* **Метрики:** Итоговая модель достигла **Mean Absolute Error (MAE) в $49.02**, что является отличным результатом, учитывая экстремальную дисперсию на рынке аренды Нью-Йорка.
* **Анализ важности признаков (Feature Importance):** 
  * *Тип жилья* и *Расстояние до центра* оказались сильнейшими предикторами цены.
  * Созданный кастомный признак *Сегмент качества* (сгенерированный через K-Means без утечки данных) показал высокую значимость, полностью оправдав применение Unsupervised Learning в пайплайне.

Вам не нужно скачивать датасеты вручную — данные автоматически подгружаются по прямой ссылке внутри ноутбука!

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/valanchick/Dynamic-Real-Estate-Pricing.git
   ```
2. Установите зависимости (если необходимо):
   ```bash
   pip install pandas numpy scipy catboost scikit-learn matplotlib seaborn
   ```
3. Откройте и запустите ноутбук:
   ```bash
   jupyter notebook airbnb_pricing_model.ipynb
   ```
